## Веб приложение для конвертации рукописного текста в печатный (фронт)

Тензор - летняя практика 2022

## Как запустить:

### `npm i`

Установка зависимостей

### `npm start`

Запустит девелопмент на localhost:3000

### `npm test`

Запустит тесты

### `npm run build`

Сделает билд

## В проекте сконфигурированы ESLint и Prettier
